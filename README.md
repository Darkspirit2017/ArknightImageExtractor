# ArknightImageExtractor
## 明日方舟 立绘提取 + 合并


## "食用"方法
1. 使用[AssetStudio](https://www.perfare.net/1194.html)提取明日方舟charpack文件
2. 将此程序放在与Texture2D同文件夹下

![alt text](./img/note.png)

3. 运行
```
python3 arknight_extractor.py
```
